# Hiraeth

Hiraeth is a lightweight Android system optimizer with a Material You styled interface. It provides simple tools to maintain device performance by managing memory, cache, and storage optimizations.

## Features

- Clear RAM by killing background processes
- Drop system caches
- Perform FSTRIM to optimize flash storage
- Simple and clean Material You interface
- Lightweight and fast execution
- Compatible with rooted Android devices

## Modules

### Clear RAM
Kills background processes and drops caches to free system memory.

### Clear Cache
Removes temporary system cache files to reclaim storage space.
Optimizes flash storage by trimming unused blocks on supported filesystems such as ext4 and f2fs.

### System Utilities
Additional optimization tools designed to keep the system responsive and stable.
